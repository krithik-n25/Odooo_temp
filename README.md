## вҡЎ The Problem VENDORA Solves

In Indian SMEs, procurement runs on WhatsApp groups, Excel sheets, and phone calls. Decisions worth lakhs are made in DMs. There is no audit trail. No accountability. No intelligence.

| вқҢ How It Works Today | вң… How VENDORA Changes It |
|---|---|
| Vendors contacted on WhatsApp вҖ” no record | Vendors invited via structured RFQs вҖ” trackable |
| Quotes compared on Excel вҖ” no intelligence | Quotes compared on a Bloomberg-style table вҖ” intelligent |
| Approvals via phone calls вҖ” no audit trail | Approvals via hold-to-confirm mechanic вҖ” accountable |
| Invoices sent via email attachments вҖ” no tracking | Invoices generated as GST-compliant PDFs вҖ” professional |
| Finance team has zero visibility until it's too late | Finance team has real-time visibility at every step |
| Post-order negotiations happen outside the system | All negotiations locked after PO вҖ” no renegotiation |

### рҹ’¬ Negotiation Thread вҖ” Replaces WhatsApp Forever

> [!IMPORTANT]
> **The #1 unprofessional behavior in Indian SME procurement is WhatsApp. VENDORA eliminates it.**

- Linear/Notion-style messages вҖ” not WhatsApp bubbles
- System messages auto-appear on key events: `"Quote revised to вӮ№97,000 В· Oct 12"`
- Thread **locks automatically** after PO is issued вҖ” no post-approval renegotiation
- Every message timestamped and audit-logged
- Manager can read the full thread before approving вҖ” full context, zero surprises
- Export as PDF вҖ” one click, full conversation for compliance teams
